# ChatProgram-Client
  * 학교 자바 프로젝트 입니다.
  * 프로젝트 명 : 채팅 프로그램
----------------

### 주요 기능
  * 윈도우 창 UI
  * 실시간 생성된 채팅방 리스트 확인
  * 1대1 혹은 다수의 사용자와 대화.
  * 여러 채팅방 동시 입장 가능
  * 원하는 닉네임으로 채팅 가능
  * 이모티콘 전송
